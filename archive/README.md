# Archive

Historical records. **Nothing in this folder is active** — don't act on it,
don't update it (except to add newly retired material). The live picture is
always the root `CLAUDE.md`.

| File | What it is |
| --- | --- |
| `HISTORY.md` | Session-by-session build log (moved out of CLAUDE.md 2026-07-16). The durable lessons were kept in CLAUDE.md; this is the full chronology. |
| `HANDOFF-2026-07-15-launch.md` | The launch-day handoff/resolution record (all items resolved; superseded by CLAUDE.md). |
| `BRANCHES.md` | Ledger of retired git branches: final SHA, what each contained, why it's dead. |
| `logo-variant-2026-07-15.png` | Rejected alternate logo from the `launch-readiness` branch, kept so the branch can be deleted losslessly. |

Convention: repo state that stops being true moves here (or gets recorded
here) rather than being deleted, so future sessions can always reconstruct
what happened. Git history is the real archive; this folder is the index to
it that survives branch deletion.
